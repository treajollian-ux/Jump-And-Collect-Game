# Jump and Collect Game

A 2D platformer game where you jump and collect coins to reach the finish line.

## Features
- Smooth character movement and jumping
- Coin collection system
- Multiple levels
- Sound effects and background music
- Responsive controls

## How to Play
- Use **A/D** or **Left/Right** arrows to move
- Press **Space** or **W** to jump
- Collect all coins to get the highest score
- Reach the finish flag to complete the level

## Installation
1. Clone this repository
2. Open in Unity 2021.3 or later
3. Add your graphics and audio files to the Assets folder
4. Build and run!

## File Structure
- `Assets/Graphics/` - Character and background images
- `Assets/Audio/` - Music and sound effects
- `Scripts/` - All C# scripts
- `Scenes/` - Unity scene files
